# Bench-Power-Supply-2021
A custom-designed bench-top power supply that converts AC to DC with adjustable output. PCB designed in DipTrace and optimized for hobbyist lab use. 

## Features
- AC-to-DC conversion with adjustable output voltage
- Custom PCB layout
- Overcurrent protection

## Hardware
- Transformer
- Rectifier diodes
- Voltage regulator ICs
- Potentiometer for voltage adjustment
- Custom PCB
- Capacitors, resistors, connectors

## Software / Tools
- DipTrace (PCB design)
- Multisim
- Soliworks
- Workshop/Power Tools for Metal Work
